# Main Features
- Display a list of NYC High Schools 
- Display all the SAT scores - including Math, Reading, and Writing.
- Error state screen


## Architecture
  - MVVM Architecture (Model - ComposableView - ViewModel)
  - Repository pattern
  - Hilt - dependency injection
